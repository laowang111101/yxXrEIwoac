# 基于Python的图书馆管理系统 python801

## 项目概述

本项目是一个基于Python开发的图书馆管理系统，整合了前端框架Vue和后端框架Django，适用于计算机设计项目学习和实践。

## 技术栈

- **开发语言**：Python
- **数据库**：MySQL
- **前端框架**：Vue
- **后端框架**：Django

## 功能模块

### 管理员功能

1. **个人中心**
2. **用户管理**
3. **图书分类管理**
4. **图书信息管理**
5. **图书借阅管理**
6. **图书归还管理**
7. **图书入库管理**
8. **信息反馈**
9. **系统管理**

### 用户功能

- 以游客身份访问：
  - 查看首页
  - 浏览图书信息
  - 阅读系统公告
  - 查看信息反馈
- 注册用户可进行：
  - 需求操作前的登录

## 系统角色

- 管理员：负责系统的管理和图书管理操作
- 用户：进行图书查询、借阅等操作

## 运行环境

- Python环境
- MySQL数据库
